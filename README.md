# API FOR DEVTREE

### Stack
- Nodejs
- Express.js
- TypeScript
- MongoDB
- Mongoose
  
### Instalation
- Clone respository
  ```
    git clone https://github.com/rody-huancas/devtree-backend.git
  ```

- Access the folder
  ```
    cd devtree-backend
  ```
  
- Install dependencies
  ```
    pnpm install
  ```

- Execute
  ```
    pnpm dev
  ```
